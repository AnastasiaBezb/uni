Выберите язык программирования

Какой язык выбрать для изучения?

Java

Серверные приложения, веб-приложения, андроид приложения, десктопные приложения

Java — сильно типизированный объектно-ориентированный язык программирования. Приложения Java обычно транслируются в специальный байт-код, поэтому они могут работать на любой компьютерной архитектуре с помощью виртуальной Java-машины. На 2019 год Java — один из самых популярных языков программирования.

Python

Веб-разработка, Data Science (машинное обучение, анализ данных и визуализация), автоматизация процессов, встроенные приложения

Python — высокоуровневый язык программирования общего назначения. Cтандартная библиотека включает большой объём полезных функций. Поддерживается разбиение программ на модули, которые, в свою очередь, могут объединяться в пакеты. В 2018 году Python поднялся на 3 место в рейтинге языков программирования Tiobe.

Перейти

C#

Десктопные приложения, веб-приложения

C# — объектно-ориентированный язык программирования. Разработан в 1998—2001 годах группой инженеров компании Microsoft как язык разработки приложений для платформы Microsoft .NET Framework. C# относится к семье языков с C-подобным синтаксисом, из них его синтаксис наиболее близок к C++ и Java.

1C

Настройка, обновление и модернизация стандартных конфигураций, возможность создавать собственную конфигурацию бухучета с нуля

1С — встроенный язык программирования 1С:Предприятия. Средой исполнения языка является программная платформа «1С:Предприятие». Визуальная среда разработки — «Конфигуратор». Наиболее подобен по своему синтаксису языку Visual Basic.
